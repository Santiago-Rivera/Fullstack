CSS Media Queries

What are Media Queries?

Media queries are a powerful feature of CSS that allow you to apply styles based on the characteristics of the device or viewport, such as its width, height, orientation, and resolution.

Basic Syntax

@media (condition) {
    /* CSS rules */
}

Example:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Common Use Cases

  • Responsive design
  • Adjusting layouts for different screen sizes
  • Changing styles for print media
