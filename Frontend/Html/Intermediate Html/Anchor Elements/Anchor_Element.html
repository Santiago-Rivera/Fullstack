Anchor Elements

The <a> element is used to create hyperlinks in HTML. It can link to other web pages, files, email addresses, or locations within the same page.

Basic Syntax

The basic syntax of an anchor element is as follows:

<a href="URL">Link Text</a>

Attributes

The <a> element has several attributes that can be used to modify its behavior:

  • href: Specifies the URL of the page the link goes to.
  • target: Specifies where to open the linked document. Common values are _blank (new tab) and _self (same tab).
  • rel: Specifies the relationship between the current document and the linked document.
  • title: Provides additional information about the link, usually displayed as a tooltip.
  • download: Indicates that the target will be downloaded when the link is clicked.
